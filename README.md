## 项目简介
LUNA-BADGE 是一个前端应用，由Gemini 3 Pro模型驱动，用来快速生成拥有工业风/瑞士风/现代/创意/商务红/极简等多种主题的个人工牌。支持离线生成赛博朋克风格的人物信息、PNG 导出、复制到剪贴板、中文/英文切换，以及轻度 3D 悬浮动效。

## 核心功能
- 多主题工牌：`Industrial`、`Modern`、`Swiss`、`Creative`、`Formal Red`、`Minimalism`
- 一键身份生成：无网络也可生成，内置赛博朋克风数据库
- 中文/英文切换：界面与字段即时切换
- 文本自适应：针对长文本自动调整字号、行数限制，避免溢出
- 导出与复制：支持 `PNG` 导出与“复制到剪贴板”
- 质感细节：玻璃拟态背景、噪声纹理、轻度 3D 悬浮旋转

## 快速开始
**环境要求：** 安装 `Node.js`

1. 安装依赖：
   `npm install`
2. 启动开发服务器：
   `npm run dev`
3. 打开浏览器访问：
   `http://localhost:3000/`（端口可能因占用自动调整，如 `3001/3002`）

## 使用说明
- 在左侧侧边栏选择“工牌风格”，实时预览不同主题效果
- 填写 `姓名 / 职位 / 工号 / 公司 / 联系方式 / 地址` 字段
- 使用“样式自定义”区域为不同主题添加头部/底部标签等文案
- 点击“生成身份”可随机生成离线赛博朋克风资料
- 点击“导出 PNG 卡片”下载高分辨率图片；或点击“复制到剪切板”直接复制图片
- 右上角“语言”按钮可在 `中文/English` 间切换

## 离线模式说明
- 项目内置本地数据集，无需任何 API Key 或网络请求
- 当网络不可用或未配置 Key 时，自动使用离线生成逻辑
- 离线生成的数据结构与在线版本保持一致，保障界面行为稳定
